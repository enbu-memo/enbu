画像ファイル採番・管理ルール

【基本ルール】
- ページごとに画像ファイル名の先頭にページ名を付ける
- 連番や用途を明記する
- 拡張子は .png, .jpg など

【例】
- ougi_01.png   ... 奥義ページ用画像1枚目
- ougi_02.png   ... 奥義ページ用画像2枚目
- raid_01.jpg   ... レイドページ用画像1枚目
- kassen_icon.png ... 合戦ページ用アイコン

【用途説明】
- ougi_01.png: 奥義ページのメイン画像
- raid_01.jpg: レイドページのバナー画像

【補足】
- 画像の説明や用途もこのファイルに追記してください
- 不明な画像は用途・ページ名を記載して管理
